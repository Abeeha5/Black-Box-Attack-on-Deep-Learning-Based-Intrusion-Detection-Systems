03 Black-Box Surrogate Model
The attacker sends 25,195 inputs to the target CNN and records only the hard predicted label (0 or 1) — no access to weights, gradients, or probability scores. A surrogate MLP is trained on these (input, label) pairs to approximate the target's decision boundary. Fidelity measures how closely the surrogate mimics the target.
Left: Surrogate–Target fidelity (agreement on test set decisions). Right: Surrogate accuracy vs. ground-truth labels.

04 Constrained PGD Attack & Transferability
PGD is run white-box on the surrogate, then adversarial examples are transferred to the target — this is the black-box threat model. Feature constraints are strictly enforced: binary flags and one-hot categoricals are never perturbed.
Mean absolute perturbation per feature. Blue bars = continuous features (allowed). Red bars = binary/categorical features (must remain zero — any non-zero value indicates a constraint violation).
Transferability heatmap. Each cell shows the Attack Success Rate (%) when adversarial examples crafted on the row model are evaluated on the column model. Diagonal = white-box; off-diagonal = black-box transfer.
t-SNE projection of attack samples (clean vs. adversarial). Separation between clusters shows how PGD shifts attack traffic toward the normal region in feature space.
