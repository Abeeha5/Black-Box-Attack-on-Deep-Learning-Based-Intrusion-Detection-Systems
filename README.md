Network Intrusion Detection Systems (IDS) built on deep learning have demonstrated near-perfect accuracy under clean traffic conditions. However, their resilience under adversarial manipulation remains poorly understood, particularly in realistic black-box threat scenarios where an attacker has no access to model internals. This project designs and evaluates a complete adversarial machine learning pipeline applied to network IDS. We train two deep learning target models a one-dimensional Convolutional Neural Network (CNN) and a Long Short-Term Memory (LSTM) network on the NSL-KDD benchmark dataset. A surrogate Multi-Layer Perceptron (MLP) is trained exclusively on input–output pairs obtained by querying the target CNN under black-box conditions (hard labels only). Constrained Projected Gradient Descent (PGD) adversarial examples are crafted on the surrogate and transferred to both target models. Two defense strategies are evaluated: feature-space clipping (input preprocessing) and adversarial training (data augmentation with synthesized adversarial examples). Results show that the CNN and LSTM both achieve 99.42% F1 on clean traffic. Surrogate-based black-box adversarial examples achieve a 4.37% Attack Success Rate (ASR) against the undefended CNN. Clip defense provides no additional benefit over the raw model for this attack budget, while adversarial training reduces ASR dramatically to 0.13% a 33× reduction without sacrificing clean detection performance (F1 remains 99.48%). Surrogate–target fidelity reaches 97.59%, validating the black-box threat model. These findings highlight that while DL-based IDS are inherently robust to mild constrained perturbations, adversarial training is a highly effective and computationally tractable defense.
